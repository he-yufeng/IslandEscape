# Island Escape Final Deliverables

This folder contains the final report and presentation working package.

## Files

- `main.tex` - LaTeX report draft. The visual rendering and animation section is intentionally reserved for Zhiyuan.
- `ppt_outline.md` - slide-by-slide plan for a 7-minute presentation.
- `speech_script.md` - live gameplay demo script for the main presenter.
- `live_demo_one_page_card.md` - one-page English cue card for the live demo.
- `live_demo_cheatsheet_zh.md` - Chinese backup cue card.
- `live_demo_preflight_checklist.md` - final run and submission checklist.

## Compile

```bash
pdflatex main.tex
pdflatex main.tex
```

If TeX is not installed locally, paste `main.tex` into Overleaf.

## Final checklist

- Use the playable game as the main presentation artifact.
- Keep PPT and video open only as backup material.
- Submit the source zip separately from report/PPT/video files.
- Do not paste API keys, tokens, cookies, or local `.env` values into deliverables.
